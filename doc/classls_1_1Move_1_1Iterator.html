Detailed Description

An iterator implementation for ls::Move.

The iterator takes an ls::Move (stored in Iterator::move) and the first flag (stored in Iterator::flag). Incrementing and dereferencing the iterator repeatedly returns all flags stored within Iterator::move. Incrementing and dereferencing further results in Move::none.
