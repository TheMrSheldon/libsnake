Detailed Description

The Move class is an iterable flag-register for the 4 directions: up, down, left, and right.

◆ opposite()

constexpr Move ls::Move::opposite ( ) const
inlineconstexprnoexcept

Returns a move instance that contains the opposite directions of this move.

Note the difference with the ~-operator which inverts the flags on the move instance: e.g. ~left is up|down|right whereas opposite(left) is right. It is also possible to get the opposite of a move which has multiple flags set: opposite(up|left) is down|right additionally, opposite(up|left|right) is down|right|left.

Returns
A new ls::Move instance pointing in the opposite direction of this move.
