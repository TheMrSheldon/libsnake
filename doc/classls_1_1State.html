Prints a board to the provided outputstream.

The function prints a box of size width × height and calls the callback to print the cell for each position. It is assumed that the callback returns exactly cellWidth characters which are then printed to the stream.

Template Parameters
FThe type of the callback. It is kept generic to allow for any callable object to be passed.
Parameters
outThe outputstream to print the state to.
widthThe width of the state.
heightThe height of the state.
callbackA callback called with a Position argument to retrieve what to print for the respective cell.
cellWidthThe width of a single state-cell.
