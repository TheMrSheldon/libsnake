DLLEXPORT SnakeFlags ls::gm::ChallengeGamemode::getWinner ( const State &  state) const
overridevirtualnoexcept

Returns the flags of all snakes that have won the game.

The returned flags may contain multiple snakes. This may indicate a tie (e.g. when both snakes win in a 1v1) but could also indicate a winning party (e.g. in Squadmode). If the game is not over #SnakeFlags::None is returned.

Parameters
stateThe state for which to return the winners.
Returns
Flags containing the flags that have won the game of the state.
