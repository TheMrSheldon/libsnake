Detailed Description

The abstract Gamemode-class should be implemented by the actual gamemodes according to their own gamerules. A gamemode may be viewed as the instance enforcing the rules by calculating the next gamestate from a previous one and determining if the game is over and who won.

Returns the move-flags of all the unblocked actions for the snake of the given index.

It is not required that all the returned moves returned are not blocked. The simplest implementation of this method could simply return all moves.

virtual SnakeFlags ls::Gamemode::getWinner ( const State &  state) const
pure virtualnoexcept

Returns the flags of all snakes that have won the game.

The returned flags may contain multiple snakes. This may indicate a tie (e.g. when both snakes win in a 1v1) but could also indicate a winning party (e.g. in Squadmode). If the game is not over #SnakeFlags::None is returned.

Parameters
stateThe state for which to return the winners.
Returns
Flags containing the flags that have won the game of the state.

virtual State ls::Gamemode::stepState ( const State &  state,
const std::vector< Move > &  moves 
) const
pure virtualnoexcept

Steps the provided state. This means that it calculates the state of play after the provided moves (moves) are played. moves contains the moves for each snake (even the dead ones) such that the move of the snake at index i in #State::getSnakes() is at index i in moves.

Parameters
stateThe state which should be stepped. The state itself will not be modified.
movesThe moves of each snake at the corresponding position in #State::getSnakes(). As such moves should have exactly the same number of arguments as #State::getSnakes().
Returns
The state after each of the moves were applied.

Implemented in ls::gm::StandardGamemode.
